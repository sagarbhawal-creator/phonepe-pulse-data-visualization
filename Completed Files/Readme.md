# PhonePe Transaction Analysis Dashboard

## Overview

This project analyzes digital payment data from the PhonePe ecosystem
using Python, SQL, Pandas, Plotly, and Streamlit.

## Features

-   Interactive Streamlit dashboard
-   Year and Quarter filters
-   KPI metrics for transactions, users, and insurance
-   Transaction trend analysis
-   Device dominance insights
-   Market expansion analysis
-   User growth visualization

## Tools & Technologies

-   Python
-   MySQL
-   Pandas
-   Plotly
-   Streamlit
-   GitHub

## Project Structure

PhonePe_Dashboard/ ├── app.py ├── database.py ├── pages/ │ └──
analysis_dashboard.py └── README.md

## How to Run

pip install streamlit pandas plotly mysql-connector-python

streamlit run app.py

## SQL Analysis

This project includes 25 SQL queries across 5 scenarios: 1. Transaction
Dynamics 2. Device Dominance 3. Insurance Growth 4. Market Expansion 5.
User Growth

## Insights

-   Digital payments are growing rapidly.
-   Some states dominate the transaction ecosystem.
-   Mobile usage is concentrated among a few brands.
-   Insurance adoption is increasing.
